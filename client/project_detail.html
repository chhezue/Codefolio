개발 기간

2023.09 - 2024.01

역할

풀스택 개발 (개인 프로젝트)

주요 기술

React Node.js MongoDB AWS

주요 기능

AI 이미지 생성

사용자의 텍스트 프롬프트를 기반으로 DALL-E 2 API를 활용하여 고품질 이미지를 생성합니다. 다양한 스타일과 해상도 옵션을 제공합니다.

갤러리 관리

생성된 이미지를 개인 갤러리에 저장하고 관리할 수 있습니다. 태그 기반 필터링과 검색 기능을 제공합니다.

기술적 도전과 해결과정

1이미지 생성 최적화

대규모 이미지 생성 요청 처리를 위해 큐 시스템을 구현하고, Redis를 활용한 캐싱 전략을 도입하여 응답 시간을 50% 개선했습니다.

2보안 강화

JWT 기반 인증 시스템 구현과 API 요청 제한을 통해 서비스의 보안성을 강화했습니다.
